В зависимости от операционной системы необходимо запускать соответствующий файл

**При парсинге ссылок необходимо включить ВПН**.\

## Запуск

### Windows
1. Переместить файл `links_parser_win` в любое удобное место
2. Запустить

### Linux
1. Переместить файл `links_parser_linux` в любое удобное место
2. Открыть терминал
3. Перейти в директорию с файлом `links_parser_linux`
4. Ввести в терминал `./links_parser_linux`
5. Нажать Enter

### Mac Os
1. Переместить файл `links_parser_mac` в любое удобное место
2. Открыть терминал
3. Перейти в директорию с файлом `links_parser_mac`
4. Ввести команду `sudo xattr -cr /путь/к/links_parser_mac`
5. Нажать на файле control + левая кнопка мыши -> открыть
6. Запустить


## Использование
* Обязательно включить VPN
* При запуске, программа попросит ввести:
** target domain - Основной домен, на котором будет происходить поиск ссылок (av.ru, stage.av.ru)
** Ключевое слово для поиска на странице (facebook, instagram, ...)

* После коррекного ввода начнется парсинг ссылок на выбранном домене (длится около 2-3 минут)
* Когда парсинг завершится, создастся файл "parsed_links/found_hrefs.txt", в котором будут указаны ссылки на страницы, где было найдено ключевое слово

## При возникновении ошибок 
* В случае, если парсинг не начинается, проверьте, подключен ли VPN
* В большинстве случаев появления ошибок - удалить все сгенерированные файлы в директории со скриптом.
